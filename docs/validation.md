# 安卓端验证清单

- 键盘打开时底部输入区是否完全可见，心情标签不遮挡输入
- 小屏（≤360dp）设备头部“更多”按钮是否可见且可点击
- 头部概览弹窗显示连续、周/月目标与进度环，内容不溢出
- Tabs 在键盘打开时是否隐藏（tabBarHideOnKeyboard）
- 热力图与统计页面标题与文案是否为中文，月份显示为“YYYY年M月”
- 个人中心各分组（提醒、数据、成就、目标、收藏标签）功能是否正常
- 通知权限申请中文提示是否正确；提醒内容为中文
- 导出 JSON/Markdown/CSV/成就摘要文件内标题与内容是否为中文
- 深色/浅色模式下对比度与可读性是否良好
- 横竖屏切换后布局是否稳定，无元素溢出屏幕

## 设备与分辨率建议
- 360×640dp（小屏安卓）
- 411×891dp（常见中大屏安卓）
- 刘海屏设备（状态栏与底部系统栏可见）

## 构建与调试建议
- app.json 已设置 android.softwareKeyboardLayoutMode 为 pan
- 如需切换为 resize，请对首页与 Tabs 进行回归测试
- CI 构建：确保 GitHub Actions 已配置 EXPO_TOKEN（Expo 账号令牌）
- 触发验证：此文档可用于轻微变更以触发 CI 构建
